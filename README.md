# Superstore Sales Analysis (Beginner Project)

Superstore Sales Analysis (Beginner Project)

This project is a simple sales analysis using the Superstore dataset.
I performed basic data cleaning, date formatting, and created simple charts using Python.

Files in this project----
train.csv – Dataset
Untitled.ipynb – Jupyter Notebook (code + graphs)

What I did----

Loaded the dataset

Checked and handled missing values

Converted the Order Date into the correct format

Analyzed sales by category and region

Created bar charts and a line plot

Explored basic sales trends

Tools Used-----

Python

Pandas

Matplotlib

Jupyter Notebook
